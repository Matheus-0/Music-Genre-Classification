## Como rodar o projeto

### Bibliotecas necessárias

Para poder rodar o projeto é preciso instalar algumas bibliotecas, para isso rode os seguintes comandos:

```
pip3 install pandas
pip3 install matplotlib
pip3 install librosa
pip3 install tqdm
pip3 install pydub
```
